FAQ


  • Combien coûte le Contrôle Parental 3G+ Xooloo pour Bouygues Telecom ?


    Le Contrôle Parental 3G+ est offert gratuitement par Bouygues Telecom aux clients Forfait Internet Mobile 3G+/EDGE.

  • Comment recevoir un numéro de licence ?


    Avant de télécharger et d'installer le logiciel, vous devez préalablement demander votre numéro de licence en inscrivant votre adresse e-mail dans le formulaire suivant :

    A savoir : le téléchargement du logiciel ainsi que ses mises à jour s'ajoutent à la consommation de votre Forfait Internet Mobile.

    Astuce : vous pouvez télécharger le logiciel sur votre ordinateur portable via une connexion fixe ou Wi-Fi. Cependant la validation de votre licence d’utilisation se fait obligatoirement via une connexion 3G+/EDGE Bouygues Telecom.

  • Comment télécharger le logiciel de Contrôle Parental 3G+ :


    Vous pouvez télécharger le logiciel de Contrôle Parental 3G+ pour PC en cliquant sur ce lien (Windows Xp et Vista).

    Vous pouvez télécharger le logiciel de Contrôle Parental 3G+ pour Macintosh en cliquant sur ce lien.

    A savoir : le téléchargement du logiciel ainsi que ses mises à jour s'ajoutent à la consommation de votre Forfait Internet Mobile.

  • Comment installer le logiciel de Contrôle Parental 3G+ ?


    Pour installer le Contrôle Parental 3G+, vous devez préalablement avoir demandé un numéro de licence, téléchargé le logiciel et enregistré celui-ci sur votre ordinateur. Nous vous conseillons de l'enregistrer sur le bureau.

    Une fois le logiciel téléchargé, double-cliquez sur l'Icône « Contrôle Parental 3G+ » présente sur votre bureau, et laissez-vous guider par les instructions du logiciel.

  • Puis-je choisir mon mode de filtrage ?


    Oui, celui-ci est modifiable en un seul clic à n'importe quel moment.

    Vous disposez de trois modes de filtrages :

    • > Le mode Enfant : Vos enfants naviguent sur une sélection de plusieurs milliers de sites Internet dont les contenus sont préalablement vérifiés par notre équipe de documentalistes. Les chats, forums, ou blogs sont explicitement exclus.
    • > Le mode Adolescent : Vos enfants naviguent sur la totalité des sites Internet à l'exception des sites identifiés comme inappropriés par le logiciel.
    • > Le mode Parent : vous permet de surfer sans aucun filtrage grâce à votre mot de passe «Parent».
     

  • Dois-je toujours être connecté avec ma connexion internet haut-débit mobile Bouygues Telecom pour utiliser le logiciel ?


    Non, le logiciel est actif quelle que soit votre connexion à Internet. Toutefois, pour demander votre numéro de licence, vous devez être connecté depuis votre connexion internet haut-débit mobile Bouygues Telecom.

  • Et si j'ai des questions additionnelles ?


    Nous vous invitons à nous faire parvenir vos questions par email à l'adresse suivante : support.logiciel3G@xooloo.net

  • Est-ce que le Contrôle Parental 3G+ fonctionne avec tous les navigateurs ?


    Oui, le Contrôle Parental 3G+ fonctionne avec n'importe quel navigateur : Microsoft Internet Explorer, Mozilla, Firefox, Opéra, Netscape...

  • De quel type d'ordinateur ai-je besoin pour utiliser le Contrôle Parental 3G+ ?


    Le contrôle parental 3G+ est compatible avec les systèmes suivants :

    Pour PC :

    Windows XP / Vista toutes versions
    Processeur Intel Pentium III à 600 Mhz ou plus
    512 Mo de RAM, 40 Mo d'espace disque disponible
    JVM Java 1.5 ou ultérieure (télécharger ici : http://java.com/fr/)

    Pour Mac :

    Mac OS X 10.4 + , 512 Mo de RAM ; 40 Mo d'espace disque disponible.
     

  • Avec quels systèmes d'exploitation le Contrôle Parental 3G+ fonctionne-t-il ?


    Pour les PC : Le Contrôle parental fonctionne avec Windows XP et Windows Vista,
    Pour les Macintosh : Le Contrôle Parental fonctionne avec Mac OS X, à partir de Mac OS 10.4.

  • Suis-je obligé de disposer d'une connexion Internet haut-débit mobile pour bénéficier du logiciel ?


    Oui, pour disposer d'un numéro de licence du Contrôle Parental 3G+, vous devez être abonné à une offre de connexion à internet haut débit mobile Bouygues Telecom.

  • Dois-je avoir une compétence spécifique pour installer et utiliser le Contrôle Parental 3G+ ?


    Non, aucune compétence spécifique n'est nécessaire à l'utilisation et à l'installation du logiciel de Contrôle Parental 3G+.

  • Est-ce que je peux facilement activer et désactiver le Contrôle Parental 3G+ ?


    Oui, il vous suffit de cliquer sur le bouton « déprotéger » et de saisir votre mot de passe « Parent ».

  • Est-ce que le Contrôle Parental 3G+ ralentit la connexion Internet ?


    Non, l'utilisation du Contrôle Parental 3G+ ne ralentit pas la connexion Internet.

  • Est-il possible d'autoriser ou d'interdire des sites soi-même ?


    Oui, il vous suffit de vous rendre dans la console d'administration et d'autoriser ou d'interdire des sites dans la catégorie « listes personnelles ».

  • Lorsque je copie le numéro de licence envoyé par e-mail, le texte « numéro de licence invalide » apparait, que faire ?


    Si ce texte apparaît, cela peut être pour l'une des raisons suivantes :

    • > l'e-mail mentionné dans le champ « e-mail » n'est pas le même que celui auquel la licence vous a été envoyée.
    • > lors de la recopie du numéro de licence, une erreur s'est produite.
     

  • Je pense que je ne suis pas filtré


    Allez sur la page de test du filtrage pour vérifier l'état de votre filtrage.

    Sinon, vérifiez que vous avez bien activé le logiciel avec une licence après l'installation et référez-vous à l'aide disponible ci-dessus.

  • J'ai oublié mon numéro de licence.


    Vous pouvez re-demander votre numéro de licence en inscrivant votre adresse e-mail dans le formulaire suivant :

    A savoir : pour re-demander votre numéro de licence, vous devez être connecté depuis une connexion internet haut débit mobile Bouygues Telecom.

  • J'ai oublié mon mot de passe « Parent »


    Lorsque vous avez oublié votre mot de passe, vous avez la possibilité de répondre à la “question en cas d'oubli du mot de passe” que vous avez entrée dans le logiciel lors de l'installation. Pour cela, trompez-vous trois fois de mot de passe : la question apparaîtra. Vous n'aurez plus qu'à y répondre correctement.

    Si vous ne vous rappelez plus de la réponse à la question secrète, vous pouvez demander à ce que le mot de passe « Parent » vous soit renvoyé par e-mail. Dans ce cas vous devez inscrire votre adresse e-mail dans le formulaire suivant. Votre mot de passe vous sera immédiatement renvoyé.

  • Comment réaliser la mise à jour du logiciel de Contrôle Parental 3G+ ?


    Les mises à jour du logiciel sont automatiques.

  • Télécharger la documentation du Contrôle Parental 3G+


    Télécharger la documentation pour PC.
    Télécharger la documentation pour Mac.
